### Meeting Request
```
Subject: Meeting Request: [Topic] - [Date/Time Range]

Hi [Name],

I'd like to schedule a meeting to discuss [topic].

Would any of these times work for you?
- [Option 1]
- [Option 2]

The meeting should take about [duration].

Best regards,
[Name]
```

### Follow-up
```
Subject: Following up: [Original topic]

Hi [Name],

I wanted to follow up on our [conversation/email] from [date].

[Brief reminder of context]

[Question or next step]

Please let me know if you have any questions.

Best regards,
[Name]
```

### Status Update
```
Subject: [Project] Status Update - [Date]

Here's the weekly update on [project]:

**Completed:**
- Item 1
- Item 2

**In Progress:**
- Item 3

**Blockers:**
- [If any]

**Next Steps:**
- Item 4

Let me know if you have any questions.

Best,
[Name]
```

## ðŸ”„ Workflow

> **Source:** [Grammarly Business Email Guide](https://www.grammarly.com/blog/category/business-email/)

### Phase 1: Draft
- [ ] **Objective**: Why am I sending this email? (To inform? To ask a question?)
- [ ] **Subject**: Summarize the topic in fewer than 5 words (`Subject: Project X Status Update`).
- [ ] **Context**: Establish context in the first sentence after "Hi [Name]," (`Regarding our meeting...`).

### Phase 2: Refine
- [ ] **BLUF**: Bottom Line Up Front - Put the main message at the beginning, don't save it for the end.
- [ ] **Tone Check**: Is it aggressive or too formal? Be `Polite` but `Direct`.
- [ ] **Formatting**: Break long paragraphs into bullet points.

### Phase 3: Finalize
- [ ] **CTA**: Is there a clear "Next Step"? (`Please review by Friday`).
- [ ] **Attachments**: Did you say "Attached" but forget to attach it? Check it.
- [ ] **Proofread**: Did you spell the name correctly? Are there typos?

### Checkpoints
| Phase | Verification                                                         |
| ----- | -------------------------------------------------------------------- |
| 1     | Does the subject line explain the content without opening the email? |
| 2     | Is the main message visible without scrolling on mobile?             |
| 3     | Does the recipient know what to do ("Reply", "Review", "Ignore")?    |
